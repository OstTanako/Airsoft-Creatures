Welcome to **Airsoft Creatures RE-CODED** official repository!

# About
Airsoft Creatures its a arcade game inspired on Space Invaders, **RE-CODED** means a remake since the original turned lost media.

**DISCLAIMER:** This game have no relation with any Airsoft-like games, its just a project made for fun.

# Modding
This project uses a own custom license, you should check-it before doing something.

**Needed Tools:**
1. Godot Engine 4.6.1 - Standart
2. Git (Recomended)

Once you installed git (or you alearly installed it), open your terminal and copy this:

```bash
git clone https://github.com/OstTanako/Airsoft-Creatures.git
```

**NOTE: Acess your destined path first.**
